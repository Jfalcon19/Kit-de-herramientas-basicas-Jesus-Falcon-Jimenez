# CCleaner

## ¿Qué hace?
Limpia archivos temporales, registros innecesarios y rastros de navegación.

## ¿Para qué sirve?
Optimiza el sistema y mejora el rendimiento general.

## ¿Cuándo usarla?
Cuando tu PC está lento o quieres liberar espacio.
