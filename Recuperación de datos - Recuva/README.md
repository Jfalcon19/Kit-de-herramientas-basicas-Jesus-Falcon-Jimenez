# Recuva

## ¿Qué hace?
Recupera archivos eliminados accidentalmente en el disco duro o dispositivos externos.

## ¿Para qué sirve?
Restaurar documentos, fotos o vídeos borrados.

## ¿Cuándo usarla?
Cuando borras archivos por error o tras un fallo del sistema.
